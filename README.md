# Rise of the Olympian (ROTO)

An action-adventure style quest-driven story that is inspired by early Greek mythology.

## Concept

Rise of the Olympian is a 2.5 action-adventure game featuring a mythological storyline. Players will explore ancient environments, battle mythical foes, and unravel the fate of the Olympians.

## Features
- Story-driven, quest-driven gamemplay
- Character movement on 3D axis with 2D characters
- Chibi-style characters, 3D low-poly terrain and textures
- Still-image cutscenes with narrative and dialogue overlay

## Tech Stack
- Game Engine: Unity
- Language: C#
- Tools: Github, Unity, GoogleDocs
- Target Platform: PC/Mac, possibly mobile
